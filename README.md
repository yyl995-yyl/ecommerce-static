# 极简电商网站 - GitHub Pages版

这是一个部署在GitHub Pages的静态电商网站，包含认证功能。

## 功能特性

- ✅ 商品列表展示
- ✅ 商品详情页面
- ✅ 分类筛选功能
- ✅ 响应式设计
- ✅ 专属访问认证
- ✅ 24小时会话保持

## 部署步骤

1. 创建GitHub仓库
2. 上传所有文件到仓库
3. 开启GitHub Pages功能
4. 访问我的网站

## 访问方式

网站地址：`https://yyl995-yyl.github.io/ecommerce-static/index.html`

首次访问需要输入密码

## 文件结构

- `login.html` - 登录页面
- `index.html` - 商品列表页
- `detail.html` - 商品详情页
- `style.css` - 样式文件

## 自定义设置

### 修改密码
在 `login.html` 文件中修改：
```javascript
const CORRECT_PASSWORD = "你的新密码";
```

### 添加商品
在 `index.html` 和 `detail.html` 的 `PRODUCTS_DATA` 数组中添加新商品。
